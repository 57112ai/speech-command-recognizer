protectRoute(req, res, next) → {void}

This middleware function: 1. Extracts the JWT token from the cookies. 2. Verifies the token using the JWT_SECRET environment variable. 3. If the token is valid, it retrieves the user from the database based on the token's payload. 4. If the user exists, it attaches the user object to the request (`req.user`). 5. If any step fails, it sends an appropriate error response.
Parameters:
Name Type Description
req express.Request The request object, which contains the cookies and other request data.
res express.Response The response object, used to send responses to the client.
next express.NextFunction The next middleware function to call if authentication is successful.
